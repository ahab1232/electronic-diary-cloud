# Electronic Diary Cloud

Информационная система для управления учебным процессом
в образовательных учреждениях.

## Цель проекта
Создание единой цифровой платформы для учета успеваемости,
расписаний, заданий и взаимодействия участников обучения.

## Пользователи
- Студент
- Преподаватель
- Администратор

## Основные функции
- Просмотр оценок и успеваемости
- Публикация домашних заданий
- Расписание занятий
- Уведомления и объявления
- Управление пользователями
- Формирование отчетов
